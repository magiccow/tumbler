# tumbler
Driving tumble dryer display from Arduino

See also Youtube video: https://youtu.be/AcXCiF9_nTw

Arduino sketches described in the video:

        light-segment-1a.ino             first example, showing 1st digit, segment a
        animate-leds.ino                 second example, animating one segment across digits
        animate_leds-2-bars.ino          two segments animated, one after the other
        lamp-test.ino                    light all 7-segment digits, then colon, sun , half-sun
        shownumber.ino                   show 4 digit number, counting up in seconds
